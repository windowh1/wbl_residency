===============================================
건강보험 임플란트 보장 기준 정보
===============================================
출처: 국민건강보험공단 관련 정보 (2025년 기준)

1. 나이 제한
---------------
• 만 65세 이상 건강보험 가입자 및 피부양자
• 적용 시점: 만 65세 생일이 지난 해의 다음 해 1월 1일부터
  예) 2025년 8월 15일 생일 → 2026년 1월 1일부터 적용 가능

2. 보장 개수
---------------
• 1인당 평생 2개까지만 건강보험 적용
• 시술 위치 제한 없음 (앞니, 송곳니, 어금니 모두 가능)
• 자연치아가 존재하지 않고 시술이 반드시 필요한 치아에만 적용
• 주의: 완전 무치악(치아가 하나도 없는 경우)은 틀니 대상으로 임플란트 보험 적용 제외

3. 본인부담금
---------------
• 본인부담금 비율: 약 30%
• 공단 부담: 약 70%
• 예상 비용:
  - 일반 임플란트 평균 비용: 120만~150만 원
  - 건강보험 적용 시 본인부담금: 약 35만~45만 원
• 주의사항:
  - 병원에 따라 추가 재료비, 진단비, 시술 전 검사비 별도 부과 가능
  - 고급 임플란트 사용 시 본인부담금 30% 이상 가능

4. 추가 정보
---------------
• 적용 제외 사례:
  - 완전 무치악 (틀니 대상으로 전환)
  - 외상·교통사고 등 (산재 또는 자동차보험 적용)
  - 심미 목적 (미용 교정 등)
• 기초생활수급자·차상위계층: 일부 본인부담금 감면 혜택 가능
• 과거 동일 치아 위치에 건강보험 적용 임플란트 시술 받은 경우: 재적용 불가
