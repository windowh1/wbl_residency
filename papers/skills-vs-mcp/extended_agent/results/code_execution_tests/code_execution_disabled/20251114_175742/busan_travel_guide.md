# 부산 여행 가이드

## 📍 부산 주요 관광지 TOP 5

## 1. 해운대 해수욕장

### 기본 정보
- **위치**: 부산 해운대구
- **특징**: 부산의 대표 해수욕장, 백사장 길이 1.5km, 폭 70-90m
- **운영시간**: 
  - 여름 개장 기간: 6월 말 ~ 8월 말 (매일 09:00 - 18:00)
  - 입수 가능 시간 및 안전요원 배치
- **입장료**: 무료

### 대중교통 접근
- **지하철**: 부산 지하철 2호선 **해운대역** 하차 → 3번 또는 5번 출구 → 도보 5분
- **버스**: 139번, 140번, 239번, 302번 등 → '해운대 해수욕장' 정류장 하차

### 편의시설
- 샤워장, 탈의실, 파라솔/튜브 대여 가능
- 주변 다양한 카페 및 식당

## 2. 광안리 해수욕장

### 기본 정보
- **위치**: 부산 수영구
- **특징**: 광안대교 야경이 아름다운 해수욕장, 젊은 분위기
- **운영시간**: 
  - 여름 개장 기간: 6월 말 ~ 8월 말 (09:00 - 18:00)
  - 연중 해변 산책 가능
- **입장료**: 무료

### 대중교통 접근
- **지하철**: 부산 지하철 2호선 **광안역** 하차 → 3번 출구 → 도보 10분
  - 또는 **금련산역** 하차 → 3번 출구 → 도보 7분
- **버스**: 24번, 63번, 139번 등 → '광안리 해수욕장' 정류장 하차

### 추천 포인트
- 야경 명소 (특히 광안대교 조명)
- 주변 카페거리 및 포차거리
- 매년 가을 부산 불꽃축제 개최지

## 3. 감천문화마을

### 기본 정보
- **위치**: 부산 사하구 감내2로 203
- **특징**: '한국의 마추픽추', 계단식 산복도로, 알록달록한 벽화마을
- **운영시간**: 매일 09:00 - 18:00 (입장 자유)
- **입장료**: 무료 (일부 전시관 및 체험 프로그램 유료)

### 대중교통 접근
- **지하철**: 부산 지하철 1호선 **토성역** 하차 → 6번 출구
  - 마을버스 사하구1-1, 서구2, 서구2-2 탑승 → '감천문화마을' 정류장 하차
- **버스**: 
  - 일반버스: 1-1번, 2번, 2-2번 마을버스
  - 사하구청이나 남포동에서 출발하는 마을버스 이용 가능

### 주요 볼거리
- 어린왕자 & 사막여우 포토존
- 하늘마루 전망대
- 골목길 벽화와 예술작품
- 다양한 카페와 기념품샵

## 4. 태종대 유원지

### 기본 정보
- **위치**: 부산 영도구 전망로 24 (동삼동)
- **특징**: 해안 절벽과 숲이 어우러진 자연 관광지, 맑은 날 대마도 조망 가능
- **운영시간**: 
  - 하절기 (3월 ~ 10월): 04:00 - 24:00
  - 동절기 (11월 ~ 2월): 05:00 - 24:00
- **입장료**: 무료
- **주차료**: 1시간 1,000원 (이후 10분당 200원, 1일 최대 10,000원)

### 대중교통 접근
- **버스**: 
  - 부산역에서: 88번, 101번 버스 탑승 → '태종대' 종점 하차 (약 30-40분)
  - 남포동에서: 8번, 30번, 66번, 88번 버스 이용
- **지하철**: 1호선 **남포역** 하차 후 버스 환승

### 추천 코스
- 다누비 관광열차 (유료) 이용 또는 도보 산책
- 태종대 전망대에서 바다 경관 감상
- 자갈밭 해안 산책로
- 신선바위, 태종사 방문

## 5. 용두산공원 & 부산타워

### 기본 정보
- **위치**: 부산 중구 용두산길 37-55
- **특징**: 부산의 랜드마크, 120m 높이 전망대에서 부산항과 영도 조망
- **운영시간**: 
  - 용두산공원: 연중무휴 24시간 개방
  - 부산타워: 10:00 - 22:00
- **입장료**: 
  - 공원: 무료
  - 부산타워: 성인 8,000원, 청소년 6,000원, 어린이 5,000원

### 대중교통 접근
- **지하철**: 부산 지하철 1호선 **남포역** 하차 → 1번 출구 → 도보 5분
  - 또는 **중앙역** 하차 → 13번 출구 → 에스컬레이터 이용
- **버스**: 
  - 남포동 정류장에서 하차 후 도보 이동
  - 주요 노선: 8번, 26번, 40번, 71번, 81번 등

### 주요 볼거리
- 부산타워 전망대에서 시내 전경 감상
- 야간 미디어파사드 및 조명쇼
- 이순신 장군 동상
- 꽃시계 및 공원 산책로

## 🚇 부산 대중교통 이용 팁

### 1. 부산 지하철
- **1호선**: 노포역 ↔ 신평역 (주요역: 서면, 남포, 자갈치)
- **2호선**: 장산역 ↔ 양산역 (주요역: 해운대, 광안, 서면)
- **3호선**: 수영역 ↔ 대저역
- **4호선**: 미남역 ↔ 안평역

**교통카드**: T-money, 캐시비 사용 가능  
**요금**: 기본 1,400원 (카드 기준)

### 2. 시내버스
- 일반버스, 급행버스, 마을버스 운영
- 교통카드 사용 시 환승 할인 적용
- 주요 관광지는 대부분 버스로 접근 가능

### 3. 관광 패스
- 부산 시티투어버스 이용 가능
- 레드라인, 블루라인 등 순환 코스 운영

## ⚠️ 여행 TIP

1. **성수기 (7-8월)**: 해수욕장은 매우 혼잡하니 이른 시간 방문 권장
2. **날씨**: 해안가는 바람이 강하니 겉옷 준비
3. **주차**: 주말/공휴일은 대중교통 이용 추천
4. **운동화**: 감천문화마을, 태종대는 계단과 경사가 많음
5. **사진 촬영**: 맑은 날 오전이 가장 좋은 빛
6. **먹거리**: 각 지역마다 특색있는 맛집과 카페가 많으니 미리 검색 권장
